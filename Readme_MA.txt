1) First open a maze file

2) Next, select a log file (either multiple or single path). If a file with multiple path is selected, all paths associated with the current maze will be loaded.

3) From the left pane, select paths (root) to display all paths

4) Or, select individual paths to display separately.

5) By default, view vector position is also enabled (so if you select a single path, you will see the path and the tip of the viewing vector that displays where the participant looked.)

6) On the right pane, "view vector display" can be set to false to hide.
